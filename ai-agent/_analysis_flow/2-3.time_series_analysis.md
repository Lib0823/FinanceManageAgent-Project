## 1. 개요

시계열 분석은 KIS API에서 수집한 120거래일 데이터를 Prophet 모델로 학습하여 **3개 피처**를 산출하는 단계다. 가격 추세와 거래량 추세를 5거래일 예측 기울기로 표현하여 Gemini가 단기 방향성을 판단하는 데 활용한다.

| 피처명 | 의미 |
| --- | --- |
| `prophet_price_trend` | D+1~D+5 가격 추세 기울기 |
| `prophet_volume_trend` | D+1~D+5 순매수 비율 추세 기울기 |
| `prophet_price_uncertainty` | 가격 예측 불확실성 강도 |

---

## 2. 설계 원칙

### 2-1. DB 저장 없음 — KIS API 직접 조회

분석 대상 종목이 매일 동적으로 바뀌는 구조에서 시계열 데이터를 DB에 누적 저장하는 방식은 관리 비용 대비 실익이 낮다. 신규 종목 진입 시 초기화 처리가 필요하고, 매일 누적/삭제 로직이 추가로 요구된다.

따라서 파이프라인 실행 시 분석 대상 종목이 확정되면 그 즉시 KIS API에서 120거래일 일봉을 직접 조회하여 메모리에 올리고, Prophet 학습·예측이 끝나면 원천 데이터는 버린다.

```
API 호출 계산:
30종목 × 2회(120일치 커버) = 60회 호출
초당 5건 제한 → 약 12초 소요 → 파이프라인 시간 내 충분히 처리 가능
```

### 2-2. 종목별 독립 학습 — 매일 새로 fit

Prophet은 단변량 시계열 모델 구조상 여러 종목 데이터를 합산하여 공통 학습하는 것이 불가능하다. 삼성전자의 가격 패턴과 다른 종목의 패턴을 섞으면 오히려 노이즈가 증가한다.

매일 파이프라인 실행 시 종목별로 독립적으로 fit하는 방식이 Prophet의 표준 사용 방법이며, 종목당 학습 시간이 수 초 이내라 30종목 전체를 처리해도 실용적인 범위 안에 있다. 학습된 모델은 당일 예측에만 사용하고 저장하지 않는다.

### 2-3. 5거래일 예측 — 기울기 피처 추출

내일 하루 예측값 하나보다 D+1~D+5의 추세 방향이 Gemini 판단에 더 유의미한 컨텍스트를 제공한다. 5개 예측값에 선형회귀를 적용하여 기울기를 산출하며, 양수면 상승 기조, 음수면 하락 기조, 절대값이 클수록 추세 강도가 강함을 의미한다.

### 2-4. 순매수 비율 — 거래량 방향성 확보

총 거래량만으로는 거래가 매수 주도인지 매도 주도인지 구분할 수 없다. KIS API 분봉 데이터에서 매수 체결량과 총 거래량을 가져와 순매수 비율을 계산하면 거래량 추세에 방향성이 담긴다. 추가 API 호출 없이 분봉 데이터에서 계산 가능하다.

---

## 3. 원천 데이터

### 3-1. 일별 종가 (KIS API)

120거래일 일봉에서 종가(close_price)를 추출한다. Prophet의 가격 계열 학습 입력으로 사용된다.

### 3-2. 일별 순매수 비율 (KIS API)

분봉 데이터에서 매수 체결량과 총 거래량을 집계하여 일별 순매수 비율을 계산한다.

**산출식**: 매수 체결량 / 총 거래량

0.5 이상이면 매수세 우위, 0.5 미만이면 매도세 우위를 의미한다. 이 값의 120거래일 추세를 Prophet으로 학습하여 거래량 계열의 방향성을 피처에 반영한다.

---

## 4. 전처리

두 계열 모두 동일한 전처리 파이프라인을 적용한다.

### 4-1. 이상값 제거

NumPy percentile 기준으로 극단값을 제거한다. 거래 정지 해제 직후 급등락, 데이터 수집 오류 등으로 발생하는 비정상적 수치가 Prophet 학습을 왜곡하지 않도록 처리한다.

### 4-2. 정렬

거래일 기준 오름차순으로 정렬한다. 공휴일·비거래일은 별도 처리 없이 건너뛰며, Prophet이 불규칙 날짜 간격을 자동으로 처리한다.

### 4-3. Prophet 입력 형식 변환

Prophet은 날짜 컬럼 이름을 `ds`, 값 컬럼 이름을 `y`로 요구한다. 거래일 컬럼을 `ds`, 종가 또는 순매수 비율을 `y`로 변환하여 입력한다.

---

## 5. Prophet 학습

### 5-1. 모델 구성

Prophet은 Meta(구 Facebook)가 개발한 시계열 예측 모델로, 추세(trend)·계절성(seasonality)·휴일 효과를 분리하여 학습한다. 불규칙한 날짜 간격을 자동으로 처리하므로 주식 거래일 기반 데이터에 적합하다.

120거래일은 약 6개월에 해당하며, 이 기간으로 학습 가능한 계절성은 다음과 같다.

| 계절성 | 학습 가능 여부 | 비고 |
| --- | --- | --- |
| 주간 (요일 패턴) | ✅ | 월요일 효과 등 |
| 월간 (월말 수급) | ✅ | 기관 리밸런싱 패턴 |
| 연간 | ❌ | 120거래일 부족 |

연간 계절성을 학습하려면 250거래일(약 1년) 이상이 필요하지만, 시장 구조 자체가 1년 전과 달라질 수 있어 오래된 데이터의 노이즈가 증가하는 트레이드오프가 있다. 120거래일은 이 균형점으로 선택된 값이며, 연간 계절성 미반영은 향후 확장 방향으로 남겨둔다.

### 5-2. 학습 방식

종목별로 독립 학습하며, 매일 파이프라인 실행 시 새로 fit한다. 학습된 모델 객체는 당일 예측에만 사용하고 저장하지 않는다.

## 6. 예측 및 피처 산출

### 6-1. 예측 범위

오늘(D+0) 기준으로 D+1부터 D+5까지 5거래일 앞을 예측한다. Prophet의 `make_future_dataframe` 함수로 미래 날짜를 생성하고 `predict`로 예측값을 산출한다.

Prophet은 각 날짜에 대해 다음 세 가지 값을 반환한다.

| 값 | 의미 |
| --- | --- |
| `yhat` | 예측 중앙값 |
| `yhat_lower` | 예측 하한 (신뢰구간 하단) |
| `yhat_upper` | 예측 상한 (신뢰구간 상단) |

### 6-2. prophet_price_trend 산출

D+1~D+5의 `yhat` 5개 값에 선형회귀를 적용하여 기울기(slope)를 추출한다.

**해석**: 양수 → 가격 상승 기조 / 음수 → 가격 하락 기조 / 절대값 클수록 추세 강도 강함

### 6-3. prophet_volume_trend 산출

순매수 비율의 D+1~D+5 `yhat` 5개 값에 선형회귀를 적용하여 기울기를 추출한다.

**해석**: 양수 → 매수세 강화 기조 / 음수 → 매도세 강화 기조

총 거래량이 아닌 순매수 비율을 사용하므로, 거래량 증가가 매수 주도인지 매도 주도인지 방향성까지 담긴다.

### 6-4. prophet_price_uncertainty 산출

D+1~D+5의 신뢰구간 폭 평균을 산출한다.

**산출식**: mean(yhat_upper - yhat_lower) for D+1 ~ D+5

**해석**: 값이 클수록 예측 불확실성이 높음. Gemini 프롬프트에서 "이 종목은 가격 예측 불확실성이 높습니다"와 같은 컨텍스트로 활용하여 판단 신뢰도를 보조한다.

## 7. 피처 정의

| 피처명 | 원천 | 산출 방법 | 해석 방향 |
| --- | --- | --- | --- |
| `prophet_price_trend` | 일별 종가 | D+1~D+5 yhat 선형 기울기 | 양수 → 상승 기조 |
| `prophet_volume_trend` | 일별 순매수 비율 | D+1~D+5 yhat 선형 기울기 | 양수 → 매수세 강화 |
| `prophet_price_uncertainty` | 일별 종가 신뢰구간 | D+1~D+5 구간 폭 평균 | 클수록 불확실성 높음 |

## 8. Gemini 입력 컨텍스트 내 위치

시계열 3개 피처는 정량 7개, 감성 1개와 합산되어 **Gemini 입력 총 11개**를 구성한다.

| 피처 그룹 | 피처 목록 |
| --- | --- |
| 정량 (KIS 기반) | morning_return, close_position, foreign_net_buy, institutional_net_buy |
| 정량 (DART 기반) | per, roe, operating_margin |
| 감성 | sentiment_score |
| **시계열** | **prophet_price_trend, prophet_volume_trend, prophet_price_uncertainty** |

시계열 피처는 정량·감성 피처가 담지 못하는 **단기 방향성과 추세 강도**를 제공한다. 특히 수급이 긍정적이고 감성도 좋지만 가격 추세가 하락 기조일 때처럼, 단기 역추세 신호를 Gemini가 포착하는 데 기여한다.

## 9. 설계 한계 및 확장 방향

| 항목 | 현재 한계 | 향후 확장 방향 |
| --- | --- | --- |

| 종목 연속성 | 매일 새로 fit으로 누적 없음 | 관심 종목 고정 구조로 변경 시 누적 학습 가능 |
| 다변량 모델 | Prophet 단변량 한계 | NeuralProphet 또는 LSTM으로 확장 가능 |
